## 7. Key Findings for lab-ui

1. **Accent bases are theme-dependent** (4 different hex per accent) — Apple iOS-style adaptive colors
2. **Neutral step 6 = base anchor** (#787880) — validates `neutral.rs` config
3. **Button uses 5 variant axes** — Type, Kind, Icon-button, Size, State — this is how semantic tokens map to component props
4. **Text styles have rich Russian descriptions** — usage guidance embedded in Figma
5. **Materials** (Light/Dark × Soft/Muted/Subtle/Elevated/Base) — likely map to background + elevation system
6. **Interactive states** (Default/Hover/Pressed) — mapped to FILL styles, not semantic tokens
